# Vanilla-typescript-boilerplate

**바닐라 타입스크립트로 컴포넌트를 만들수 있는 환경 boilerplate 입니다.**

## 설치 방법
```cli
yarn install
yarn start # development
yarn build # production build
```

## 설치되어 있는 개발 환경

- `typescript`
- `webpack`
- `webpack-dev-server`
- `webpack-cli`
- `babel`
- `eslint`
- `prettier`

## ESLint

### plugins

- `@typescript-eslint`
